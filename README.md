# 微积分冒险 · Calculus Adventure

> 一个将数学分析核心概念变成小游戏的 HTML5 网页应用。大一工科数学分析课程的期末作业，尽力做得有趣了一点。

## 这是什么

把极限、导数、积分、向量场四个抽象概念做成了能玩的关卡游戏。初衷是"如果当年有人拿这个给我看，我可能学得更快一点"。

22 个关卡 + 自由探索模式，纯前端单文件，打开即玩。

| 模式 | 数学概念 | 怎么玩 |
|------|---------|--------|
| 🎯 极限挑战 | ε-δ 定义、函数极限 | 拖动滑块让 x 逼近目标点，观察函数值变化趋势，选出极限值 |
| 📈 导数射手 | 导数的几何意义 | 观察切线的斜率，判断导数值 |
| 📊 积分收集 | 黎曼和与定积分 | 调整区间和矩形数量，让近似值逼近精确积分 |
| 🌀 向量场 | 梯度、旋度、散度 | 自由切换四种向量场模式，悬停查看任意点的向量 |



## 特性

- **纯前端**：单个 HTML 文件，无依赖、无构建、离线可用
- **关卡解锁制**：22 个关卡难度递进，通关自动解锁下一关
- **多种游戏机制**：生命值系统、计时挑战、连击奖励、积分星星评级
- **Canvas 动画**：ε-δ 逐帧可视化、切线扫描动画、黎曼矩形收敛动画
- **存档系统**：localStorage 本地存档，支持 JSON 导出/导入
- **成就系统**：8 个隐藏成就等待发现
- **响应式**：桌面和移动端都能玩

## 快速开始

浏览器打开 `微积分冒险游戏.html` 即可。

```
git clone https://github.com/Sirius1906/calculus-adventure.git
cd calculus-adventure
# 双击 微积分冒险游戏.html 或拖入浏览器
```
