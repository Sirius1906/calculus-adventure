## 📋 项目概述

**微积分冒险**是一款将工科数学分析四大核心概念 — **极限、导数、积分、向量场** — 转化为交互式游戏与可视化实验的 HTML5 网页应用。通过"玩中学"的方式，帮助理解抽象的数学原理。

### v2.0 新增功能

| 功能 | 说明 |
|------|------|
| ❤️ 生命值系统 | 每关3条命，答错扣命，0命重新挑战 |
| ⏱️ 计时挑战 | 限时完成关卡获取额外时间奖励分 |
| ⭐ 星星评级 | 积分游戏根据矩形效率和精度评1-3星 |
| 🔥 连击系统 | 连续首次答对累积combo，跨关卡保持 |
| 🎬 ε-δ动画 | 极限通关后展示ε缩小时δ随之缩小的动画 |
| 💾 存档导入导出 | JSON格式导出/导入游戏进度 |
| 🌟 浮动分数 | 答对时从按钮飘出"+N"分数动画 |
| 📳 屏幕震动 | 答错时游戏卡片震动反馈 |

### 核心功能

| 游戏模式 | 数学概念 | 交互方式 |
|---------|---------|---------|

## 🎮 游戏机制：关卡解锁制

每个游戏都采用 **关卡解锁制**：
- **第1关**默认解锁
- 通关当前关卡后 **自动解锁下一关**
- **首页** 可随时查看总体完成进度（总得分、已通关数、完成百分比）
- **关卡目录** 显示每个关卡的锁闭/解锁/已完成状态及最高分
- **已解锁的关卡可以随时重玩**，争取更高分数

### 导航切换

页面顶部有 6 个导航标签：

```
[🏠 首页] [🎯 极限挑战] [📈 导数射手] [📊 积分收集] [🌀 向量场] [📖 说明]
```

- **首页**：显示总体进度概览，可快速进入各个游戏
- 每个游戏页默认显示 **关卡目录**，点击已解锁关卡开始游戏

### 🎯 游戏1：极限挑战

**目标：** 判断函数在某个点处的极限值。

**操作步骤：**
1. 拖动下方的滑块，让 x 逐渐逼近目标值 x₀
2. 观察左侧画布中红色点（当前值）向黄色点（极限值）的逼近过程
3. 观察"误差"数值逐渐减小
4. 从四个选项中选择正确的极限值
5. 回答正确后点击"下一关"继续

**数学原理：**
- 极限的 ε-δ 定义：当 x 无限接近 x₀ 时，f(x) 无限接近某个确定值 L
- 回答正确后画布会显示 ε 邻域，帮助理解 ε-δ 语言

**包含关卡（8关）：**
1. `f(x) = (x²-1)/(x-1)` 在 x=1 处（可去间断点）
2. `f(x) = sin(x)/x` 在 x=0 处（重要极限）
3. `f(x) = (cos(x)-1)/x` 在 x=0 处
4. `f(x) = (1+x)^(1/x)` 在 x=0 处（自然常数 e）
5. `f(x) = (e^x-1)/x` 在 x=0 处
6. `f(x) = ln(1+x)/x` 在 x=0 处
7. `f(x) = (√(1+x)-1)/x` 在 x=0 处
8. `f(x) = tan(x)/x` 在 x=0 处

### 📈 游戏2：导数射手

**目标：** 根据函数图像上某点的切线，选出正确的导数值。

**操作步骤：**
1. 观察画布中蓝色曲线（函数）和黄色直线（切线）
2. 粉色点是切点，显示其坐标
3. 观察切线是向上倾斜（正斜率）还是向下倾斜（负斜率）
4. 切线越陡，导数的绝对值越大
5. 从四个选项中选择正确的导数值
6. 答对可累积连击（combo）获得额外加分

**数学原理：**
- 导数的几何意义：f'(x₀) = 函数在 x₀ 处切线的斜率
- 切线斜率 = Δy/Δx = 导数值

**包含关卡（8关）：**
1. `f(x) = x²` → f'(x) = 2x
2. `f(x) = sin(x)` → f'(x) = cos(x)
3. `f(x) = e^x` → f'(x) = e^x
4. `f(x) = ln(x+2)` → f'(x) = 1/(x+2)
5. `f(x) = x³-3x` → f'(x) = 3x²-3
6. `f(x) = cos(x)` → f'(x) = -sin(x)
7. `f(x) = √(x+1)` → f'(x) = 1/(2√(x+1))
8. `f(x) = arctan(x)` → f'(x) = 1/(1+x²)

### 📊 游戏3：积分收集者

**目标：** 调整黎曼和的参数，使近似面积接近精确值。

**操作步骤：**
1. 拖动"左端点 a"滑块调整积分下限
2. 拖动"右端点 b"滑块调整积分上限
3. 拖动"矩形数量"滑块增加或减少矩形数（1~50）
4. 观察蓝色矩形（黎曼和）如何逼近曲线下的面积
5. 红色半透明区域表示精确面积
6. 观察"近似值"与"精确值"之间的"误差"
7. 当误差小于容差时点击"检查答案"过关

**数学原理：**
- 定积分 ∫ₐᵇ f(x)dx = 曲线 y=f(x) 与 x 轴之间的面积
- 黎曼和：将区间 [a,b] 分成 n 个子区间，用矩形面积之和近似
- 当 n → ∞ 时，黎曼和 → 定积分精确值

**关键发现：** 增加矩形数量 n 可以显著减小误差！

**包含关卡（6关）：**
1. `f(x) = x²` 在 [0, 2] 上
2. `f(x) = sin(x)` 在 [0, π] 上
3. `f(x) = e^x` 在 [0, 1] 上
4. `f(x) = 2x+1` 在 [0, 3] 上
5. `f(x) = x³` 在 [-1, 1] 上
6. `f(x) = cos(x)` 在 [0, π/2] 上

### 🌀 游戏4：向量场探索

**目标：** 自由探索不同类型向量场的几何结构。

**操作步骤：**
1. 点击底部按钮切换向量场模式：
   - **梯度场** ∇f：向量从原点向外辐射
   - **旋转场**：向量绕原点旋转
   - **源/汇场**：类似电荷产生的电场
   - **鞍点场**：一个方向吸引，另一方向排斥
2. **鼠标悬停**在画布上，右上角信息框显示当前点的坐标和向量值
3. **点击**可固定显示某点的信息
4. 观察箭头的**方向**（表示向量方向）和**长度**（表示向量大小）

**数学原理：**
- 向量场：空间中每一点对应一个向量
- 梯度场：标量变化最快的方向
- 旋度：描述旋转程度（旋转场旋度 ≠ 0）
- 散度：描述源/汇（源场散度 ≠ 0）

## 🎯 新游戏机制

### 生命值系统
- 每个关卡开始时拥有 **3条命** (❤️❤️❤️)
- 答错一次扣1条命，0条命时自动重新挑战
- 通关时剩余生命越多，分数加成越高（每条+10分）
- 仅剩1条命通关解锁成就「命悬一线」💀

### 计时挑战模式
- 在游戏关卡控制栏点击「计时」按钮开启
- 极限挑战: 30秒 | 导数射手: 45秒 | 积分收集: 60秒
- 剩余时间越多，时间奖励分越高（最高+50分）
- 时间耗尽扣1条命并重新挑战
- 计时模式下通关累计5次解锁「时间管理大师」⌛

### 连击系统
- 连续首次答对（无错误尝试）累积combo
- combo跨关卡保持，答错后重置
- 极限挑战: combo × 15分 | 导数射手: combo × 25分
- 在分数面板实时显示当前连击数

### 积分星星评级
- 积分收集者游戏使用多维度评分：
  - 基础分 50 + 矩形效率 (≤40) + 精度分 (≤30) + 生命加成 + 时间加成
- 总分 ≥ 100: ⭐⭐⭐ | ≥ 75: ⭐⭐ | < 75: ⭐
- 星星评级在关卡目录中显示

### ε-δ 动画可视化
- 极限关卡通关后自动播放
- ε从1.0逐步缩小到0.05，δ同步缩小
- 金色横带 = ε邻域，蓝色竖带 = δ邻域
- 直观展示"对任意ε，存在δ"的核心思想

## ❓ 常见问题

**Q：打开后页面空白怎么办？**
A：请使用最新版 Chrome / Edge 浏览器，并确保 JavaScript 未被禁用。

**Q：如何导出/导入游戏进度？**
A：在「说明」标签页中点击「导出存档」下载 JSON 文件，点击「导入存档」选择之前导出的文件即可恢复进度。

**Q：生命值耗尽怎么办？**
A：生命值耗尽（0条命）后会自动重新挑战当前关卡，生命值重置为3。利用剩余生命获取额外分数加成。

**Q：计时模式怎么用？**
A：进入游戏关卡后，在底部控制栏点击「计时」按钮开启。在限定时间内答对可获得最高50分的时间奖励，超时扣一条命。

**Q：如何获得3星评级？**
A：在积分收集者游戏中，使用较少矩形（提高矩形效率分）和更小误差（提高精度分）来获得更高总分，100分以上 = 3星。

**Q：滑块拖动没有反应？**
A：点击滑块后左右拖动即可，或直接在滑块轨道上点击。

**Q：可以修改游戏内容吗？**
A：可以。用文本编辑器打开 HTML 文件，找到 `g1Levels`、`g2Funcs`、`g3Levels` 等数组即可修改关卡配置。

**Q：如何重置游戏分数？**
A：在「说明」标签页中点击「重置所有进度」按钮，或在浏览器中清除站点数据。
